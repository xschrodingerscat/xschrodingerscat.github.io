Github Pages and Hexo for Personal Bolog Deployment

On maxox platform

1 Register github account

2 Install nodejs, git and hexo

node-v16.14.2.pkg : nodejs pkg

Hexo: npm install hexo-cli -g

Git: brew install git

3 Init hexo and preview home page

Home page address: http://localhost:4000/

4 Configuration

Crate a repository having naming type xschrodingerscat.github.io

Config _config.yml in your blog directory and append the following codes:

Token : Github User -> Settings -> Developer Settings -> Personal access tokens -> Create new token (Allow all options)

Install git tool for hexo: npm install hexo-deployer-git --save

Generate static pages and deploy blog to remote github repository. hexo generate && hexo deploy

Upload : Github User -> Settings -> SSH GPG keys -> New SSH key -> Past the key just generated

Github Pages : .github.io -> Settings -> Pages -> Select the branch and folder (default docs) and Save

5 Deployment

Deploy blog to github rebository : hexo g -d or hexo generate && hexo deploy

Pernal website address: https://xschrodingerscat.github.io

6 Theme Configuration

Select the appropriate theme form Hexo theme (Chic)

Clone: git clone https://github.com/Lemonreds/hexo-theme-Chic.git themes/Chic

Less: npm install hexo-renderer-less --save

Feed: npm install hexo-generator-feed --save

Json-content: npm install hexo-generator-json-content --save

Tag : hexo new page tag

Modify _config.yml
Modify themes/Chic/_config.yml

More details, please refer to Chic

Appendix : Useful manual

hexo docs

Hexo Clean

hexo clean

Generate ssh-key for github and upload for deploying failure

Generate: ssh-keygen -t rsa (For example: all iterm to be entered are default)
Tow files will be generated in the ~/.ssh/ directory

Copy : `pbcopy < ~/.ssh/id_rsa.pub
